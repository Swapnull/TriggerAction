# Programming Test

## Task
Create a Trigger and Action based application which allows the building of Applets to perform an Action based on the state of a Trigger.

**Trigger:** receives an input and decides whether a conditional has been met
**Action:** perform a task
**Applet:** glue which joins a Trigger to an Action

#### Use Cases
1. When leaving work I want to send a text message to my SO to let them know I'll be home shortly.
2. Every Monday at 9pm I want to send an email to myself to remind me to put the bins out.
